# My Velog Archive

Velog에 작성한 기술 블로그 글과 Python 코딩테스트 학습 기록을 모아둔 저장소입니다.

## Contents

- `programmers-python/`: 프로그래머스 문제 풀이, 테스트 코드, 문제별 노트, Velog 포스팅 초안
- `posts/velog/`: 실시간 채팅 백엔드, 콘서트 예매 시스템, 오픈소스 기여 등 Velog 기술 글 초안
- `velog-31-posts-feedback.md`: 기존 Velog 글 피드백 정리용 문서

## Study Notes

- 코딩테스트 문제는 Python으로 풀이합니다.
- 풀이마다 원래 작성한 코드와 개선 풀이를 비교합니다.
- Velog 글에는 문제 요약, 접근 방식, 코드, 시간복잡도, 배운 점을 함께 정리합니다.
