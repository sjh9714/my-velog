# [DAY1] [프로그래머스/Python] 가운데 글자 가져오기 - 인덱스와 슬라이싱으로 풀기

## 문제 요약

문자열 `s`가 주어졌을 때 가운데 글자를 반환하는 문제다.

문자열의 길이가 홀수라면 가운데 한 글자를 반환하고, 짝수라면 가운데 두 글자를 반환하면 된다.

예를 들어 `s = "abcde"`라면 가운데 글자인 `"c"`를 반환한다.
`s = "qwer"`처럼 길이가 짝수라면 가운데 두 글자인 `"we"`를 반환한다.

## 처음 생각한 접근

가운데 위치를 찾기 위해 먼저 문자열 길이를 2로 나눈 몫을 구했다.

```python
mid = len(s) // 2
```

Python의 인덱스는 0부터 시작하기 때문에, 길이가 홀수인 문자열에서는 이 값이 정확히 가운데 인덱스가 된다.

반대로 길이가 짝수인 문자열에서는 가운데가 한 글자가 아니라 두 글자다.
이때는 `mid - 1`부터 `mid`까지 잘라내면 된다.

## 내가 작성한 코드

```python
def solution(s):
    mid = len(s) // 2

    if len(s) % 2:
        return s[mid]

    return s[mid - 1:mid + 1]
```

## 코드 설명

`mid = len(s) // 2`는 문자열의 가운데 근처 인덱스를 구하는 코드다.

문자열 길이가 홀수라면 `len(s) % 2`가 참이므로 `s[mid]`를 반환한다.

```python
s = "abcde"
mid = 5 // 2  # 2
s[2]          # "c"
```

문자열 길이가 짝수라면 가운데 두 글자를 반환해야 한다.
그래서 `s[mid - 1:mid + 1]`처럼 슬라이싱한다.

```python
s = "qwer"
mid = 4 // 2  # 2
s[1:3]        # "we"
```

## 좋았던 점

이번 풀이는 조건이 문제 설명과 거의 그대로 대응되어서 읽기 쉽다.

- 홀수 길이면 한 글자
- 짝수 길이면 두 글자

이렇게 코드가 문제의 요구사항과 비슷한 모양을 가지고 있어서, 나중에 다시 봐도 의도를 이해하기 쉽다.

## 개선해볼 수 있는 코드

조건문을 없애고 슬라이싱 범위를 공식처럼 계산할 수도 있다.

```python
def solution_refactored(s):
    return s[(len(s) - 1) // 2:len(s) // 2 + 1]
```

## 개선 코드 설명

이 코드는 시작 인덱스와 끝 인덱스를 길이에 따라 자동으로 맞춘다.

홀수 길이인 `"abcde"`를 예로 들면 다음과 같다.

```python
(len(s) - 1) // 2  # 2
len(s) // 2 + 1    # 3
s[2:3]             # "c"
```

짝수 길이인 `"qwer"`에서는 다음처럼 동작한다.

```python
(len(s) - 1) // 2  # 1
len(s) // 2 + 1    # 3
s[1:3]             # "we"
```

다만 처음 작성한 코드도 충분히 좋다.
코딩테스트에서는 짧은 코드보다 실수 없이 읽히는 코드가 더 중요할 때가 많기 때문이다.

## 테스트 케이스

| s | expected |
|---|----------|
| `"abcde"` | `"c"` |
| `"qwer"` | `"we"` |
| `"a"` | `"a"` |
| `"ab"` | `"ab"` |
| `"hello"` | `"l"` |
| `"python"` | `"th"` |

## 시간복잡도

시간복잡도는 `O(1)`이다.

문자열 전체를 순회하지 않고 가운데 위치만 계산해서 인덱싱하거나, 최대 두 글자만 슬라이싱하기 때문이다.

## 공간복잡도

공간복잡도도 `O(1)`이다.

반환하는 문자열의 길이가 최대 2이기 때문에 입력 크기에 비례하는 추가 공간을 사용하지 않는다.

## 배운 점

- 문자열 길이의 홀짝에 따라 반환해야 하는 범위가 달라질 수 있다.
- Python 슬라이싱은 끝 인덱스를 포함하지 않는다.
- `s[mid - 1:mid + 1]`처럼 범위를 직접 쓰면 짝수 길이의 가운데 두 글자를 깔끔하게 가져올 수 있다.
- 조건문 풀이가 더 읽기 좋을 때도 있다.

## 최종 코드

```python
def solution(s):
    mid = len(s) // 2

    if len(s) % 2:
        return s[mid]

    return s[mid - 1:mid + 1]
```

## 다음에 비슷한 문제를 만나면 떠올릴 신호

- 문자열의 가운데를 묻는 문제는 먼저 `len(s) // 2`를 기준 인덱스로 잡자.
- 길이가 홀수면 한 글자, 짝수면 `mid - 1`부터 `mid + 1`까지 슬라이싱하면 된다.
- 슬라이싱은 끝 인덱스를 포함하지 않는다는 점을 같이 떠올리자.

## 마무리

이번 문제는 인덱싱과 슬라이싱의 기본기를 확인하기 좋은 문제였다.

처음 작성한 풀이처럼 홀수와 짝수를 나눠서 처리하면 문제 조건이 그대로 드러난다.
이런 작은 문제일수록 무조건 짧게 쓰기보다, 왜 이렇게 자르는지 설명할 수 있는 코드로 작성하는 연습이 중요하다고 느꼈다.
