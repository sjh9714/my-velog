# Programmers Python Coding Test Study

Python으로 프로그래머스 코딩테스트 문제를 풀고, 매일 Velog에 풀이 과정을 기록하는 저장소입니다.

## 목표

- 8주 동안 프로그래머스 문제 꾸준히 풀기
- Lv.1 기본 구현 문제 빠르게 정리하기
- Lv.2 주요 유형 익히기
- 오답노트 작성하기
- 매일 Velog 포스팅하기

## 폴더 구조

```text
programmers-python/
├── README.md
├── progress.md
├── templates/
│   ├── blog-template.md
│   └── problem-note-template.md
├── lv1/
│   └── sum-between-two-integers/
│       ├── solution.py
│       ├── test_solution.py
│       └── README.md
└── posts/
    └── velog/
        └── 001-sum-between-two-integers.md
```

- `README.md`: 저장소 소개, 목표, 학습 규칙을 정리합니다.
- `progress.md`: 8주 동안의 문제 풀이와 블로그 작성 현황을 기록합니다.
- `templates/`: 매일 사용할 Velog 글 템플릿과 오답노트 템플릿을 보관합니다.
- `lv1/`: 프로그래머스 Lv.1 문제 풀이 코드, 테스트 코드, 문제별 정리를 보관합니다.
- `posts/velog/`: Velog에 게시할 Markdown 글 초안을 보관합니다.

## 학습 규칙

- 문제를 풀기 전에 입력 크기와 시간복잡도를 먼저 확인한다.
- 틀린 문제는 당일, 2일 뒤, 1주 뒤에 다시 푼다.
- Velog 글에는 문제 요약, 접근 방식, 코드, 시간복잡도, 배운 점을 반드시 포함한다.
