# 문자열 내 p와 y의 개수

## 문제 요약

문자열 `s`가 주어졌을 때 `p`의 개수와 `y`의 개수가 같은지 확인한다.
대소문자는 구분하지 않으며, `p`와 `y`가 모두 하나도 없는 경우에도 `True`를 반환한다.

## 접근 방식

먼저 문자열 전체를 소문자로 변환했다.
그다음 `count()`를 사용해서 `p`의 개수와 `y`의 개수를 각각 세고, 두 값이 같은지 비교했다.

## 핵심 코드

```python
text = s.lower()
return text.count('p') == text.count('y')
```

## 시간복잡도

O(n)

## 공간복잡도

O(n)

`lower()`가 소문자로 변환된 새 문자열을 만들기 때문이다.

## 배운 점

문자열에서 대소문자를 무시해야 한다면 먼저 `lower()` 또는 `upper()`로 기준을 통일하면 좋다.
개수를 비교하는 문제는 직접 반복문을 쓰기 전에 `count()`로 간단하게 표현할 수 있는지 확인해보자.
