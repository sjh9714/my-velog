# 정수 내림차순으로 배치하기

## 문제 요약

정수 `n`의 각 자릿수를 큰 수부터 작은 수 순서로 정렬한 새 정수를 반환한다.

## 접근 방식

처음에는 정수를 문자열로 바꾼 뒤, 각 문자를 리스트에 하나씩 담았다.
그다음 리스트를 내림차순으로 정렬하고, 정렬된 문자를 다시 문자열로 이어 붙여 정수로 변환했다.

개선 풀이에서는 `sorted()`와 `join()`을 사용해서 같은 흐름을 더 간결하게 표현했다.

## 핵심 코드

```python
return int("".join(sorted(str(n), reverse=True)))
```

## 시간복잡도

O(d log d)

`d`는 정수 `n`의 자릿수다.
자릿수를 정렬하는 데 `O(d log d)`가 걸린다.

## 공간복잡도

O(d)

문자열로 변환한 자릿수와 정렬된 문자 리스트를 사용하기 때문이다.

## 배운 점

- 정수의 각 자릿수를 다루려면 먼저 `str(n)`으로 바꾸면 쉽다.
- `sort(reverse=True)` 또는 `sorted(..., reverse=True)`로 내림차순 정렬할 수 있다.
- 여러 문자열을 합칠 때는 반복문에서 `+=`를 사용하기보다 `join()`을 쓰면 더 깔끔하다.
