# 가운데 글자 가져오기

## 문제 요약

문자열 s가 주어졌을 때 가운데 글자를 반환한다.
문자열의 길이가 짝수라면 가운데 두 글자를 반환한다.

## 접근 방식

문자열 길이를 2로 나눈 몫을 가운데 인덱스로 사용했다.
길이가 홀수라면 가운데 인덱스의 한 글자를 반환하고, 길이가 짝수라면 가운데 인덱스 바로 앞부터 가운데 인덱스까지 두 글자를 슬라이싱한다.

## 핵심 아이디어

- `mid = len(s) // 2`
- 홀수 길이: `s[mid]`
- 짝수 길이: `s[mid - 1:mid + 1]`

## 시간복잡도

O(1)

## 공간복잡도

O(1)

## 배운 점

Python 문자열은 인덱싱과 슬라이싱으로 필요한 구간만 간단하게 꺼낼 수 있다.
짝수/홀수 조건을 나눠도 충분히 읽기 좋은 풀이가 될 수 있고, 익숙해지면 슬라이싱 범위 자체를 공식처럼 정리할 수도 있다.
