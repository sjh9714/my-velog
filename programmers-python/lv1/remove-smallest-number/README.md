# 제일 작은 수 제거하기

## 문제 요약

정수 배열 `arr`에서 가장 작은 수를 제거한 배열을 반환한다.
가장 작은 수를 제거한 뒤 배열이 비어 있다면 `[-1]`을 반환한다.

## 접근 방식

처음에는 `sorted(arr)`로 정렬된 배열을 만들고, 그 첫 번째 값을 최솟값으로 사용했다.
그다음 원본 배열에서 최솟값의 위치를 찾아 `pop()`으로 제거했다.

개선 풀이에서는 정렬하지 않고 `min()`으로 최솟값을 찾은 뒤, 리스트 컴프리헨션으로 최솟값이 아닌 원소만 새 리스트에 담았다.

## 핵심 코드

```python
min_value = min(arr)
return [num for num in arr if num != min_value]
```

## 시간복잡도

O(n)

`min(arr)`가 배열을 한 번 순회하고, 리스트 컴프리헨션도 배열을 한 번 순회한다.
따라서 전체 시간복잡도는 O(n)이다.

## 공간복잡도

O(n)

최솟값을 제거한 새 리스트를 만들기 때문이다.

## 배운 점

- 최솟값만 필요할 때는 전체 정렬보다 `min()`을 먼저 떠올리자.
- `pop()`은 원본 리스트를 직접 수정한다.
- 원본을 건드리지 않고 결과를 만들고 싶다면 리스트 컴프리헨션을 사용할 수 있다.
