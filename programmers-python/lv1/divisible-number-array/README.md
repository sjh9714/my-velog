# 나누어 떨어지는 숫자 배열

## 문제 요약

정수 배열 `arr`와 정수 `divisor`가 주어졌을 때, `arr`의 원소 중 `divisor`로 나누어 떨어지는 값을 오름차순으로 반환한다.
나누어 떨어지는 값이 하나도 없다면 `[-1]`을 반환한다.

## 접근 방식

먼저 배열을 오름차순으로 정렬했다.
그다음 정렬된 배열을 순회하면서 `divisor`로 나누어 떨어지는 값만 결과 배열에 추가했다.

개선 풀이에서는 `sorted()`와 리스트 컴프리헨션을 사용해서 정렬과 조건 필터링을 더 간결하게 표현했다.

## 핵심 코드

```python
answer = [num for num in sorted(arr) if num % divisor == 0]
return answer if answer else [-1]
```

## 시간복잡도

O(n log n)

정렬에 `O(n log n)`이 걸리고, 배열을 한 번 순회하는 데 `O(n)`이 걸린다.
따라서 전체 시간복잡도는 `O(n log n)`이다.

## 공간복잡도

O(n)

정답 배열에 조건을 만족하는 원소를 담기 때문이다.
개선 풀이의 `sorted(arr)`도 새 리스트를 만든다.

## 배운 점

- `arr.sort()`는 원본 리스트를 직접 정렬한다.
- `sorted(arr)`는 원본을 바꾸지 않고 정렬된 새 리스트를 반환한다.
- 조건에 맞는 값만 모을 때는 반복문과 `append()`를 먼저 떠올릴 수 있고, 이후 리스트 컴프리헨션으로 더 간결하게 바꿀 수 있다.
