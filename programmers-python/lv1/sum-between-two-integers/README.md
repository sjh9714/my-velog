# 두 정수 사이의 합

## 문제 요약

두 정수 a, b가 주어졌을 때 a와 b 사이에 있는 모든 정수의 합을 구한다.

## 접근 방식

처음에는 두 수 사이의 거리 abs(a - b)가 홀수인지 짝수인지에 따라 경우를 나눴다.
하지만 등차수열의 합 공식을 사용하면 조건문 없이 더 간단하게 풀 수 있다.

## 핵심 공식

항의 개수 = abs(a - b) + 1
합 = 항의 개수 * (첫 항 + 마지막 항) // 2

## 시간복잡도

O(1)

## 공간복잡도

O(1)

## 배운 점

Python에서 /는 float 나눗셈이고 //는 정수 나눗셈이다.
조건문으로 경우를 나누기 전에 하나의 공식으로 통합할 수 있는지 생각해보자.
